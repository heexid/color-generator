## ✨ Overview

Color Generator is a lightweight web application that helps designers and developers create consistent color palettes from one or multiple base colors.

Perfect for:

- 🎨 Design Systems
- 🖥 UI Design
- 📱 Mobile Apps
- 🌐 Websites
- 🎯 Brand Guidelines

Instead of manually creating shades, simply add your colors and let the generator build a complete palette.

---

## 🚀 Features

- 🎨 Multiple color support
- ✏️ Rename colors anytime
- 🌈 Built-in color picker
- 🔢 Edit HEX values
- ⚡ Auto apply after selecting colors
- 📋 Copy HEX value
- 🗑 Remove colors
- 💾 Export generated palette
- 🌙 Clean modern interface
- 📱 Responsive layout

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/heexid/color-generator.git
```

Go to project

```bash
cd color-generator
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

---

## 📖 Usage

### Add a Color

Click **Add Color**, choose a color using the picker, and it will automatically generate your palette.

### Edit a Color

Every color can be updated:

- Name
- HEX value
- Color picker

Changes are applied automatically.

### Export

Export the generated palette for your project.

## 💡 Why Color Generator?

Design systems often require consistent color scales (50–950), but generating them manually is time-consuming.

Color Generator helps you:

- Maintain visual consistency
- Speed up design workflow
- Generate scalable color systems
- Prepare colors for design tokens
