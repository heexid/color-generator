# 🎨 Color Generator

A simple web app to generate and manage color palettes for design systems and UI projects.

## Features

- Generate color palettes from custom colors
- Add multiple colors
- Edit color name and HEX value
- Color picker with auto-apply
- Copy color values
- Clean and responsive interface

## Getting Started

Clone the repository:

```bash
git clone https://github.com/heexid/color-generator.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
